<h1>Creating a Scene</h1>

To start the project, we wanted to create the basic game objects for visualization. We added a yellow sphere as the tennis ball that was scaled 
down to the actual size (6.7 cm accoring to some googling). We added a green plane as the court scaled to the actual size (23.77x10.97m). Finally
we added a white plane beneath the court for contrast.

Additionally we experminted with built-in physics that come with the RigidBody component in Unity for the sphere. We used the built-in gravity and
added an initial velocity and direction.

Sadly we didn't take any pictures but we will use these basic objects in the future so it will probably look similar.
